Silabus

Basic

  • UI Excel, format & data validation
  • Referensi sel, aritmatika, text & date
  • Lookup dasar (XLOOKUP/VLOOKUP)
  • Sort & Filter, tabel terstruktur

Intermediate

  • Cleaning data (TRIM, CLEAN, Power Query ringan*)
  • PivotTable/PivotChart
  • IF, IFS, SWITCH, INDEX–MATCH
  • Conditional formatting tingkat lanjut

Advance

  • Dashboard (desain, chart, slicer)
  • Model data, relasi, formula array
  • Automasi ringan: makro/rekorder
  • Praktik proyek mini (UMKM/office)
